  • DevOps:

    • Operations: Systeem Beheerders
  • Commando's:
    • ls (laat zien welke bestanden en folders in de huidige locatie zitten)
    • cd Folder/ (gaat naar de gegeven folder toe)
    • cd .. (gaat terug naar de vorige folder)
    • git init (maakt een repository aan in huidige folder)
    • rm .git -r (verwijdert de git map)
    • git add . (voegt bestanden toe aan repository)
    • git commit -m "commit enz." (commit naar Git)
    • git remote add origin https://github.com/apexdk/HelloWorld.git (connect met GitHub repo)
    • git push -u origin master (verstuurt naar GitHub repo)
